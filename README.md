# idkDOE

Python library for design of experiments workflows in engineering simulation and optimization projects.

## What It Provides

- Utilities for defining experiment spaces
- Sampling workflows for simulation studies
- Project structure for integrating DOE outputs with downstream modeling and optimization tools

## Installation

```bash
pip install .
```

## Status

Early-stage framework extracted from engineering simulation workflows. The public repository is intended as a portfolio/reference implementation and may require project-specific configuration before reuse.
